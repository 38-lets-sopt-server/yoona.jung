# 자유 게시판 목록
## 광고 목록 전체 조회(get)
### 응답
    - 광고 이미지
    - 제목
    - 앱이름
    - 앱 설명 텍스트

## 고정 게시글 조회(get)
### 요청
    - 게시판 아이디
### 응답
    - 고정 게시글 개수
    - 고정 게시글 제목
    - 고정 게시글 내용

## 전체 게시글 조회(get)
### 요청
    - 게시판 아이디
### 응답
    - 게시글 제목
    - 게시글 미리보기 텍스트
    - 댓글 수
    - 조회 수
    - 시간
    - 익명 여부

## 인기 게시글 조회(get)
### 요청 
    - 게시판 아이디
### 응답
    - 인기 게시글 제목
    - 인기 게시글 공감 수

# 게시글 상세
## 게시글 상세 조회(get)
### 요청
    - 게시글 아이디
### 응답
    - 사용자 아이디 (익명 여부 반영)
    - 사용자 프로필
    - 게시글 작성 시간
    - 게시글 제목
    - 게시글 내용
    - 공감 개수
    - 댓글 개수
    - 스크랩 개수

## 댓글 전체 조회(get)
### 요청
    - 게시글 아이디
### 응답
    - 사용자 프로필
    - 사용자 아이디 (익명 여부 반영)
    - 댓글 내용
    - 댓글 작성 시간

## 댓글 작성(post)
### 요청
    - 게시글 아이디
### 응답
    - 내용
    - 익명 여부

# 글 쓰기
## 게시글 작성(post)
### 요청
    - 제목
    - 내용
    - 익명 여부
    - 이미지
    - 파일
    - 질문 여부
### 검증
    - 제목 최대 50자 제한
    - 제목 null 불가
    - 이미지 최대 10장 제한
